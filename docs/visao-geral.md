# LopesCore – Visão Geral

O LopesCore é um projeto de portfólio desenvolvido a partir de uma necessidade real do dia a dia de um supermercado.

Trabalhando diretamente no setor de açougue, foi possível observar que muitos produtos acabam indo para quebra por falta de visibilidade, organização e acesso rápido à informação.

A proposta do LopesCore não é substituir sistemas já existentes, mas oferecer uma camada de apoio mais clara, organizada e fácil de acessar para o controle de quebras, perdas e movimentações internas.

O foco inicial do projeto é validar a ideia através de um MVP simples, funcional e bem estruturado, simulando um cenário real de uso dentro de um supermercado.
